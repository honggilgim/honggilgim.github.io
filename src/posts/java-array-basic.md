## 배열

배열은 **같은 타입의 여러 변수를 하나의 묶음으로 다루는 것**이다. 같은 타입이어야 하고 서로 다른 타입은 배열로 묶을 수 없다.

## 배열의 선언

```java
타입[] 변수이름;
타입 변수이름[];
```

의 형식으로 선언된다.

```java
int[] arr = new int[5];
int arr[] = new int[5];
```

두 가지의 형식이 있고 더 선호하는 것을 사용하면 된다. 배열을 선언한 후에는 꼭 뒤에 new를 통해 형식과 길이를 지정해주어야 한다. 배열을 생성하는 것은 단순히 배열을 다루기 위한 참조변수를 위한 공간을 만드는 것이고 배열을 생성해야만 값을 저장할 수 있는 공간을 생성하는 것이다.

### 배열의 생성 과정

1. `int[] arr;`

   int 형 배열의 참조변수를 선언한다. 값을 저장할 수는 없다.

2. `arr = new int[5];`

   메모리의 빈 공간에 int형 데이터를 저장할 수 있는 다섯 개의 공간을 생성한다. (arr[0] ~ arr[4] 까지 사용할 수 있다.)

## 배열의 길이와 인덱스

생성된 배열의 각 저장공간을 배열의 요소라고 하며, `배열이름[인덱스]`의 형식으로 배열의 요소에 접근이 가능하다. 인덱스는 배열의 요소마다 붙여진 일련번호로 배열을 구별할 때 사용한다.

위의 배열 `int arr[] = new int[5];`의 경우, arr[0]에서 arr[4]까지 인덱스가 존재한다.

이 인덱스에는 상수 대신 변수나 수식도 사용이 가능하다.

범위를 벗어난 인덱스는 `ArrayIndexOutOfBoundsException`을 발생시킨다. (컴파일로 못잡는다.)

배열의 길이는 0 ~ 20억까지 가능한데 0의 경우는 나중에 사용할 일이 생긴다.

`배열.length` 함수는 배열의 길이를 리턴한다.

## 배열의 초기화

```java
int arr[] = new int[] {10, 20, 30, 40, 50};
```
