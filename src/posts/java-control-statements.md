## 개요

프로그램의 흐름을 변경할 수 있는 구문이 있는데, 우리는 이를 제어문이라고 한다. 제어문에는 조건문과 반복문이 있는데 조건문은 조건에 따라 다른 문장이 수행되도록 하고, 반복문은 특정 문장들을 반복해서 실행한다.

## 조건문

if, switch 문장이 있다.

### if 문

조건문이 참일 경우 아래의 구문을 수행한다.

조건문은 보통 비교연산자와 논리 연산자로 구분된다. 조건문의 연산 결과는 항상 true 또는 false로 나타나야 한다.

### if-else 문

if문의 변형 구문이다. if 뒤에 else 구문이 새로 추가되었으며 이 구문은 조건문이 참이 아닐 때 그 문을 실행한다.

### if-else if ... else 문

if 뒤에 다른 조건을 붙이는 구문이다.

```java
if(조건문) {
} else if(조건문) {
} else if(조건문) {
} ... 
else {
}
```

의 형태를 가진다.

## switch 문

switch 문은 if문과 달리 하나의 구문으로 많은 경우의 수를 처리할 수 있고, 표현도 간결하여 알아보기 쉽다. if 문 분기가 많다면 switch구문을 사용하는 일이 권장되지만 제약조건이 많다.

```java
switch (조건문) {
    case 값:
        // 구문
        break; // switch 벗어남
    case 값:
        // 구문
        break; // switch 벗어남
    default:
        // case 문을 하나도 안 탈 때 이동
}
```

break 구문이 중요한데, break가 없을 경우 case의 구분을 진행하지 않아 case 이후에도 구문이 진행된다. 경우에 따라 고의적으로 break를 구분하지 않아 사용하기도 하니 이 점은 알아두어야 한다.

### 제약조건

switch 문의 조건식은 조건식 결과가 정수 또는 문자열이어야 한다. case 문의 값은 정수 상수만 가능하여 문자열도 불가능하고, 중복되지 않아야 한다.

## 반복문 - for, while, do-while

반복문은 어떤 작업이 반복적으로 진행될 때 사용한다. for와 while은 구조적으로 동일하며, for 문은 주로 반복의 횟수를 알고 있을 때 사용한다.

## for문

for문은 반복 횟수를 알고 있을 때 주로 사용하며

```java
for(초기화; 조건문; 증감식) {
}
```

의 형태로 주로 사용된다.

```java
for(int i = 0; i < 5; i++)
```

이런 형태로 주로 사용된다. 조건문이 값이 참이면 반복을 계속하고, 거짓이면 반복을 멈춘다.

### 향상된 for 문

JDK 1.5부터 추가된 기능이다. 배열, 컬렉션 등에 저장된 변수에 접근할 때 기존보다 편리한 방식으로 접근이 가능하다.

```java
for(타입 변수명 : 배열 또는 컬렉션)
```

의 형식으로 이루어진다.

```java
List<HashMap> list = new ArrayList<>();

... // 할당

for(HashMap a : list) {
    a.get("key"); // a 값 사용 가능
}
```

이러한 방식으로 사용이 가능하다.

## while 문

while 문은 간단한 조건문을 통해 조건문이 참일 경우 거짓이 될때까지 반복한다.

```java
int i = 0;

while(i < 5) {
    System.out.println(i);
    i++;
}
```

이런 식으로 사용한다.

## do-while 문

for 문과 while 문은 조건문이 참일 경우에만 동작한다. 하지만 do-while 문은 최소 한번의 실행을 전제로 한다.

```java
do {
    // 구문
} while (조건문);
```

이런 식으로 사용된다. 많이 안쓰이는 구문이라 한번도 못봤다.

## break 문

while 문, for 문을 포함한 모든 반복문에 들어갈 수 있다. break가 걸릴 경우 그 자리에서 반복문을 빠져나온다.

## continue 문

반복문에서 사용될 경우 반복문의 끝으로 이동해 다음 반복으로 이동한다.

## 이름 붙은 반복문

반복문에 이름을 붙일 수 있다. break, continue 문을 사용할 수 있는데 이렇게 할 경우 이름 붙은 반복문을 중지 혹은 진행시킬 수 있다.

```java
Loop: 
for(int i = 0; i < 5; i++) {
    for(int j = 0; j < 5; j++) {
        // 반복문
        if(j == 5)
            break Loop;
        if(j == 1)
            continue Loop;
    }
}
```

이런 형식으로 사용이 가능하다. 반복문에 Loop라는 이름을 붙여, continue와 break 구문에 사용했다.
