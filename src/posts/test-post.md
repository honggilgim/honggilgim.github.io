# 테스트 포스트

이것은 마크다운으로 작성된 테스트 포스트입니다.

## 소개

React와 마크다운을 사용하여 블로그 포스트를 작성할 수 있습니다.

## 기능

- **마크다운 지원**: 일반적인 마크다운 문법을 모두 지원합니다
- **코드 블록**: 코드를 예쁘게 표시할 수 있습니다
- **이미지**: 이미지도 삽입할 수 있습니다

## 코드 예제

다음은 JavaScript 코드 예제입니다:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
