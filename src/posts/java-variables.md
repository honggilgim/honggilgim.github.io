## 변수란

### 정의

변수란 값을 저장할 수 있는 메모리상의 공간을 의미한다. 하지만 단 하나의 값만을 저장할 수 있다.

### 선언

자바에서 변수는 변수타입 + 변수이름으로 형성된다. 변수타입은 저장하고자 하는 값의 종류에 맞게 변수의 타입을 선택해서 적어주고 자바는 정수형, 실수형, 문자형 등 다양한 타입을 제공한다. 변수의 이름은 메모리 공간에 이름을 정하는 거다. 같은 이름의 변수는 여러 개 존재해서는 안 되고 서로 구별될 수 있어야 한다.

### 대입 연산자 =

변수를 사용하고 처음 사용할 때, 변수는 적절한 값으로 초기화 시켜주어야 한다. 이 때 사용하는 것이 `=`이다.

```java
int a = 123;
```

a라는 int 형 변수에 123을 할당했다.

변수의 두 값을 바꾸기 위해서는 메모리 공간이라는 점을 잘 생각해서 바꾸어야 한다.

```java
int x = 1;
int y = 2;
int temp = x;
x = y;
y = temp;
```

의 형식으로 바꾸어야 이상없이 값이 변경된다.

### 자바 변수 명명규칙

자바의 변수의 이름은 식별자(identifier)라고 한다. 이 식별자는 항상 서로 구분되어야 한다. 또한, 식별자를 만들 때는 규칙을 지켜야 한다.

1. 대소문자 구분 및 길이 제한 X
2. 예약어(true와 같은 문자) 사용 금지
3. 숫자로 시작 X
4. 특수문자는 오직 `_`와 `$`만 사용 가능

변수의 이름은 식별자로 잘 지켜야 한다.

## 타입

프로그래밍 언어에서 주로 사용하는 값은 종류에 따라 크게 문자와 숫자로 나눌 수 있으며 숫자는 정수와 실수로 나눌 수 있다. 이러한 값의 종류에 따라 크기 및 저장형식을 정해주는 부분을 자료형이라고 한다.

문자형(char), 정수형(byte, short, int, long), 실수형(float, double)

### 기본형과 참조형

자바의 자료형은 형식에 따라 크게 기본형과 참조형으로 나누어 질 수 있다.

**기본형 변수는 실제 값(data)를 저장하고, 참조형 변수는 어떤 값이 저장되어 있는 주소를 값으로 가진다.** 자바는 참조형 변수 간의 연산을 지원해주지 않아, 실제 연산에 사용되는 값들은 모두 기본형 변수이다.

1. **기본형(primitive type)**
   - 논리형(boolean), 문자형(char), 정수형(byte, short, int, long), 실수형(float, double)

2. **참조형 (reference type)**
   - 객체의 주소를 저장한다. (String, Integer 등)

참조형 변수는 기본 변수형이 아닌, 클래스의 이름을 사용한다. 새로운 클래스를 작성하는 것 역시 새로운 참조형 변수를 선언한다고 생각하면 된다. 참조형은,

```java
String s; // 클래스이름 변수이름
```

처럼 선언된다.

String과 같은 참조형 객체는 자주 사용되는 객체라 `=`으로 선언 및 할당이 가능하다. 하지만, 일반 참조형 변수는 다른 방식으로 선언해 주어야 한다.

```java
Date today = new Date();
```

참조형 변수의 선언 방식이다. Date 타입의 객체를 생성해서 참조변수 할당값인 주소를 객체에 저장한다.

## 기본형

기본형은 총 4개의 타입을 가진다.

### 1. 논리형

**boolean**

true와 false의 값을 가지며 조건식과 논리적 계산에 사용된다.

### 2. 문자형

**char**

문자를 저장하는데 사용하며, 한 char 변수 하나당 한 가지의 문자만 저장이 가능하다.

### 3. 정수형

**byte, short, int, long**

정수를 저장하는 데 사용하며 주로 int를 사용한다. byte는 이진 데이터를 저장한다. CPU 효율성은 int, 길이는 long, 메모리는 byte, short에 특화되어 있다.

### 4. 실수형

**float, double**

실수를 저장하는 데 주로 사용한다. float은 짧은 실수(7자리의 정밀도를 가진다.) double은 15자리의 정밀도를 가진다.

char는 문자형 데이터지만, 정수 유니코드를 사용하기 때문에 정수형 또는 실수형과 연산도 가능하다. boolean을 제외한 모든 변수형은 서로 연산이 가능하다.

## 상수와 리터럴 변수

### 상수

상수는 변수와 마찬가지로 값을 저장하지만 변경이 불가능하다.

```java
final int MAX_NUMBER = 123;
```

변수 앞에 final을 붙이면 된다. 이 변수값은 변경이 안되고, 선언 즉시 초기화해야 한다.

### 리터럴

리터럴은 값 그 자체를 의미하는 용어이다. 상수는 값 그 자체가 아닌, 값을 한번 저장하고 변경할 수 있는 저장공간이다.

- 변수: 하나의 값을 저장할 수 있는 **저장공간**
- 상수: 변경할 수 없는 **하나의 값**
- 리터럴: 값 그 자체

### 리터럴과 접미사

리터럴은 접미사가 존재한다. 특히, 실수 혹은 정수형을 저장할 때 접미사는 중요하다. 정수형은 L, 실수형은 f, d만 알면 된다. 16진수는 0x 또는 0X, 8진수는 0이 붙는다. (JDK 1.7부터는 `_` 또한 지원된다.)

```java
// ex
// 2진수, 8진수, 16진수
int n2 = 0b10;
int n8 = 010;
int n16 = 0x10;

// 구분자 예시
long big = 100_000_000_000L;
long hex = 0xFFFF_FFFF_FFFF_FFFF;

// 실수형 예시 (대소문자 상관없음, double은 생략 가능)
float a = 2.143f;
double b = 2.13543d;
```

가끔 실무에서 쓰인다. (안쓰면 에러난다)

큰 타입의 변수는 작은 타입의 변수를 포용할 수 있다.

### 문자 리터럴과 문자열 리터럴

'A'는 문자 리터럴이고(char), "ABC"는 문자열 리터럴(String)이다.

작은 따옴표는 문자고, 큰 따옴표는 문자열이다.

문자열은 `""`은 빈 값을 허용하지만 문자 `''`은 빈 값을 허용하지 않는다.

### 지시자

간단히 C언어를 생각하면 된다. 자바에도 println()과 같은 출력함수 말고 printf()라는 함수가 존재한다. 이걸 사용하면 지시자 형식으로 사용이 가능하다. (잘 사용하지 않는다.)

```java
System.out.printf("%d%d\n", 12, 13);
// 1213후 줄바꿈으로 출력
```
